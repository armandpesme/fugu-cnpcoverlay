# Runtime paths

## Logs runtime Modrinth

Lire les logs runtime dans :

```text
%APPDATA%\ModrinthApp\profiles\[à définir]\logs
```

## Crash reports Modrinth

Lire les rapports de crash dans :

```text
%APPDATA%\ModrinthApp\profiles\[à définir]\crash-reports
```

Ne jamais déduire ou inventer lorsque l’information existe dans ces dossiers.
